# 制作の背景

二回目のポートフォリオ作成にあたり、前回の反省点を踏まえ、機能は絞り込みつつも、継続的にアップデート可能な形で制作しました。
授業で習得した技術に加え、自身で調査した内容も最大限活用し、復習と応用を重視しています。
今回は、実務で活用しやすいシンプルな設計を目指し、開発を進めました。
